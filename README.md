# Masterpiece

> 1개월 내 플레이 가능한 **싱글플레이 액션 RPG**를 만드는 것을 목표로 합니다.  
> 멀티플레이는 **아키텍처만 고려**하고 실제 구현은 후순위로 둡니다.

---

## 프로젝트 목표
- 플레이어가 성장/파밍/퀘스트를 반복하며 최종 아티팩트 **Masterpiece**를 획득하는 경험 구현
- 30~60분 분량의 완결된 1차 플레이 루프(MVP) 확보
- 이후 멀티 확장 가능한 데이터 중심 구조 유지

---

## 코어 루프
1. 마을에서 퀘스트 수락/정비/상점 이용
2. 필드 또는 던전 진입
3. 전투 및 자원 획득 (골드, 부산물, 상자)
4. 아티팩트 획득/장착/교체
5. 캐릭터 강화 및 다음 난이도 도전
6. 메인 스토리 진행 → Masterpiece 단서 수집

보조 루프:
- 반복 퀘스트
- 상자 개봉 랜덤 보상
- 던전 파밍 기반 빌드 최적화

---

## 입력/조작 (v0.1 결정안, 디아블로식)
- 이동: **마우스 우클릭**
- 기본 공격(콤보): **마우스 좌클릭**
- 스킬: **Q / W / E / R**
- 아이템 사용(소모품): **1~4 숫자키**
- 상호작용: **F**
- 회피(선택): **Space**

> 디아블로식 조작 감각을 우선 적용하며, 키 리맵 옵션은 추후 추가합니다.

## 첫 개발 캐릭터 (v0.1 우선)

### 블레이드 댄서 (근접 단일 특화)
- 컨셉: **공격 속도가 매우 빠르고 기본 공격력이 높은 단일 딜러**
- 강점: 보스/엘리트 1:1 교전, 빠른 콤보 누적, 짧은 빈틈
- 약점: 광역 처리 효율 낮음, 포지셔닝 실패 시 리스크 큼
- 기본 전투 시작점: 게임 시작 직후에는 **기본 공격(좌클릭)만 사용 가능**

> 확장 캐릭터(레인저/스위퍼 등)는 v0.2 이후 추가를 목표로 함.

---

## 전투 시스템 요약

### 기본 공격 콤보
- 좌클릭 연타로 콤보 연결
- 콤보 끊김: 시간 초과, 피격 경직, 회피 입력
- 블레이드 댄서는 기본 3타 콤보 + 마지막 타격 피해 계수 상승

### 스킬 시스템
- 슬롯: Q/W/E/R
- 자원: 쿨다운 + 마나(또는 스태미나)
- 액티브/패시브 혼합 구성
- 레벨업 시 획득한 포인트로 트리형 스킬 투자

---

## 레벨 디자인 (간단 v0.1)

### 레벨 규칙
- 시작 레벨: 1
- 최대 레벨(1차): 30
- 레벨업 보상: **스킬 포인트 1**
- 포인트 사용처: 출혈 / 깡딜 / 공속 트리
- 해금 규칙: **같은 계열의 하위 스킬에 누적 투자 시 상위 스킬 해금**

### 구간별 플레이 경험 목표
- **Lv1~5 (튜토리얼 구간)**
  - 기본 공격만으로 전투 학습
  - 첫 액티브 스킬 1개를 빠르게 해금하도록 유도
- **Lv6~15 (빌드 분기 시작)**
  - 3개 계열 중 1~2개 집중 투자
  - 보스전에서 단일 화력/생존 방식 차이 체감
- **Lv16~25 (시너지 완성 구간)**
  - 액티브-패시브 조합으로 콤보 루프 고도화
  - 던전에서 빌드 장단점이 명확히 드러남
- **Lv26~30 (엔드 직전 구간)**
  - 핵심 상위 노드 활성화
  - Masterpiece 획득 전 최종 세팅 단계

## 스킬트리 시스템 (던파식 구조 참고)

### 공통 규칙
- 트리는 3계열: **출혈 / 깡딜 / 공속**
- 각 트리는 T1 → T2 → T3 구조
- 상위 티어 해금 예시
  - T2: 해당 트리 누적 3pt
  - T3: 해당 트리 누적 7pt
- 액티브/패시브가 섞여 있음
- 스킬 레벨업도 포인트로 가능 (최대 레벨은 스킬별 별도 설정)

### 1) 출혈 트리 (지속 피해 특화)
- **T1 패시브: 혈흔 각인**
  - 기본 공격이 낮은 확률로 출혈 부여
- **T1 액티브: 절개**
  - 빠른 2연 베기, 출혈 상태 대상 추가 타격
- **T2 패시브: 응고 파괴**
  - 출혈 중인 적에게 치명타 피해 증가
- **T2 액티브: 붉은 연타**
  - 단일 대상 다단히트, 출혈 스택 즉시 폭발 일부 반영
- **T3 액티브: 혈계 처형**
  - 출혈 스택 비례 큰 단일 피해 + 남은 출혈 시간 연장

### 2) 깡딜 트리 (순간 폭딜 특화)
- **T1 패시브: 약점 관통**
  - 기본 공격 고정 피해 보너스
- **T1 액티브: 파쇄격**
  - 짧은 돌진 후 강한 일격
- **T2 패시브: 치명 압축**
  - 치명타 확률 소폭 증가, 치명타 피해 크게 증가
- **T2 액티브: 낙하 베기**
  - 단일 고계수 타격, 적 경직 강화
- **T3 액티브: 필살 일섬**
  - 긴 쿨다운의 초고배율 단일기

### 3) 공속 트리 (DPS/콤보 지속 특화)
- **T1 패시브: 가속 신경**
  - 공격 속도 증가
- **T1 액티브: 플래시 스텝**
  - 짧은 거리 순간 이동 + 다음 기본 공격 강화
- **T2 패시브: 연쇄 반응**
  - 일정 콤보 이상 유지 시 추가 공속/이속 버프
- **T2 액티브: 무영난무**
  - 짧은 시간 동안 기본 공격 히트 수 증가
- **T3 패시브: 오버드라이브**
  - 공속 상한 확장 + 강화 상태 유지 시간 증가

### 추천 빌드 예시
- **출혈 암살형**: 출혈 7pt 우선 → 깡딜 보조
- **정통 단일 폭딜형**: 깡딜 7pt 우선 → 공속 보조
- **초고속 콤보형**: 공속 7pt 우선 → 출혈 보조

## 월드 구성
- **마을(Hub)**: NPC, 퀘스트, 상점, 장비 정비
- **필드**: 중간 난이도, 안정적 재화 수급
- **던전**: 높은 난이도, 높은 보상 밀도

---

## 아티팩트 시스템

### 슬롯
- 무기 아티팩트
- 장비 아티팩트
- 악세서리 아티팩트

### 등급
- 일반
- 레어
- 유니크
- 레전더리
- 에픽
- 마스터피스

### 규칙
- 아티팩트는 계열(세트/테마)을 가짐
- 동일 슬롯 내 계열 중복 장착 불가(고유장착)
- 상위 등급일수록 수치/고유 효과 강화

### 랜덤 드랍 철학
- 사용 중인 캐릭터 전용 장비만 드랍되지 않음
- 사용하지 않는 장비 드랍도 허용(빌드 다양성, 차기 캐릭터 육성 유도)

## 아티팩트 획득처
1. 퀘스트 클리어 보상 상자
2. 몬스터 처치 시 낮은 확률 상자 드랍
3. 던전 클리어 확률 보상
4. 상점 저등급 직접 구매(골드)
5. 상점 랜덤 상자 구매(골드)
6. 대장간 제작(몬스터 부산물 조합) — 리소스 상황에 따라 v0.2 이관 가능

---

## 퀘스트 구조
- 메인: Masterpiece 단서 수집/보스 해금
- 서브: 사냥/수집/전달
- 반복: 골드/상자 파밍

메인 내러티브 뼈대:
- Act 1: 마을 주변 이상 현상 조사
- Act 2: 던전 심층 파편 수집
- Act 3: 최종 시련에서 Masterpiece 획득

## 1개월 개발 로드맵

### Week 1 — 조작/전투 코어
- 이동/카메라/점프
- 기본 콤보 프레임
- 레벨/경험치/스킬포인트 시스템 기초
- 스킬트리 UI 뼈대 + 3계열 T1 구현
- 적 AI 기본(추적/공격/피격)

### Week 2 — 월드/콘텐츠 골격
- 마을/필드/던전 1개씩 연결
- NPC 대화 + 퀘스트 수락/완료
- 드랍 테이블 + 상자 오브젝트
- 스킬트리 T2 노드 구현

### Week 3 — 성장/경제
- 인벤토리 + 아티팩트 장착
- 등급/계열 효과 적용
- 상점(직접 구매/랜덤 상자)
- 스킬트리 T3/밸런스 1차

### Week 4 — 폴리싱
- UI 정리(체력/쿨다운/인벤토리/스킬트리)
- 타격감/사운드 강화
- 버그 픽스 + 간단 세이브
- 튜토리얼 및 첫 30분 플레이 검증

## MVP 기준
- 플레이어블 캐릭터 1종(블레이드 댄서) 완성
- 마을 1 / 필드 1 / 던전 1
- 메인 퀘스트 5개 + 반복 퀘스트 3개
- 아티팩트 등급/장착/드랍/상점 동작
- 스킬트리 3계열(T1~T3) + 액티브/패시브 적용
- 30~60분 플레이 가능한 진행 구조

- 인벤토리/퀘스트 상태 동기화 가능한 저장 구조 유지
